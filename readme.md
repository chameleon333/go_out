# kousatu

## アプリ概要
アニメ、映画、漫画、小説などの**自身の考察**をシェア、意見を交換するアプリです。
ユーザーをフォローしたり、いいねなどができ、お気に入りの考察を探すことができます。

## 作成した経緯
僕自身、小説や映画などが好きで見終わったあとによく考察などを見ます。
ですが、毎回「〜 考察」と検索するのも手間で、またいろんな方の考察を比較しにくいと感じ、できるだけその手間をなくすため「kousatu」を作成しました。
